# Deep Learning Architectures and Optimization Study

This repository presents a comprehensive exploration of multiple deep learning models, focusing on architecture design, optimization techniques, and practical applications across domains like computer vision and time-series forecasting.

---

## 📌 Overview

The project investigates how different neural network architectures behave under various configurations, optimizers, and training strategies. It also integrates explainability techniques to interpret model decisions, making it suitable for research-oriented learning and experimentation.

---

## 🧠 Models Covered

### 🔹 Convolutional Neural Networks (CNN)
Implemented a custom CNN for image classification tasks with feature map visualization to understand hierarchical feature extraction.

### 🔹 VGG & AlexNet Comparison
Comparative analysis between VGG-style and AlexNet-style architectures using performance metrics such as accuracy and training loss trends.



---

### 🔹 Fully Connected Neural Networks (FCNN)
Exploration of deep feedforward networks with:
- Optimizer comparison (SGD, Adam, RMSprop)
- Mini-batch evaluation strategies
- Performance benchmarking



---

### 🔹 Recurrent Neural Networks (RNN)
Time-series forecasting model for climate prediction using sequential learning and regression-based evaluation metrics.



---

### 🔹 VGG with Data Augmentation
Enhanced CNN model using advanced augmentation techniques to improve generalization and robustness in image classification tasks.
